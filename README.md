# Marine Plastic Pollution Detection

## Problem
Detect marine plastic pollution from ocean images using deep learning.

## Dataset
Marine Plastic Pollution Dataset (Kaggle)  
https://www.kaggle.com/datasets/surajit651/souvikdataset

## Model
- CNN with Transfer Learning (VGG16)
- Data augmentation
- Binary classification

## Results
- Achieved reliable classification performance on validation data

## Tools
- Python
- TensorFlow / Keras
- NumPy
